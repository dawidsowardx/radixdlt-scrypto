Struct radix_engine::types::radix_engine_common::math::decimal::Decimal

pub struct Decimal(pub BnumI256);
Expand description

Decimal represents a 256 bit representation of a fixed-scale decimal number.

The finite set of values are of the form m / 10^18, where m is an integer such that -2^(256 - 1) <= m < 2^(256 - 1).

Unless otherwise specified, all operations will panic if underflow/overflow.
